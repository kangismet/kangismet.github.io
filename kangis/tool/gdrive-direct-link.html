Catatan:

  • Pastikan di Google Drive file sudah "Anyone with the link" (Siapa saja yang meliki link). Apabila pengaturan "Dibatasi" maka hanya Anda pemilik file yang bisa mengakses file. (Buat umum, karena tujuan akan dibagikan).
  • Tool ini hanya berjalan untuk file yang diupload, bukan dokumen di Google Docs/Sheets/Slides.
  • Apabila file terlalu besar, direct link akan mebuka halaman dan menyatakan file terlalu besar untuk scan virus. Halaman tersebut akan menampilkan tombol untuk mendownload file.

Cara Membuat File Google Drive untuk Dibagikan

Step 1: Kunjungi Google Drive kilik kanan pada file yang akan dibagikan, dan klik "Share" / "Bagikan"
Step 2: Pada jendela yang muncul, ganti akses ke "Anyone with the link" / "Siapa saja yang memiliki link"
Step 3: Klik "Copy link" / "Salin link"
Step 4: Paste link tersebut di isiian paling atas pada Tool ini, kemudian "Buat Direct Link". Salam!
